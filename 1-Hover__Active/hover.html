Hover

El hover se utiliza para hacer un "cambio de estado" cuando el mouse se posiciona sobre un objeto.

Para aplicarle estilos de hover a un objeto, tenemos que usar la pseudo-clase :hover

Ej: 

      
        .btn:hover {
          background-color: orange;
        }

Aplicando :hover
