Active

La pseudo-clase :active se activa cuando el usuario pulsa con el cursor del mouse sobre un elemento.

Para aplicarle estilos de active a un objeto, tenemos que usar la pseudo-clase :active similar a como usamos para el hover.

Ej: 

      
        .btn:active {
          background-color: green;
        }
